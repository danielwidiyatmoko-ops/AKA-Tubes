Ini sebab sering gunanya linked list dalam berbagai bidang dari memory management hingga storage management sebab linked list efisien dalam memory usagenya

ada 5 algoritma yang dilihatkan dalam sorting linked lists:
Bubble Sort,
Insertion Sort,
Selection Sort,
Merge Sort, dan
Quick Sort.

Kita memiliki dataset sendiri yang dibuat dengan bantuan excel dengan fungsi RANDARRAY() untuk menghasilkan random number sebanyak 10,100,1000,dan 10000 nomor.
